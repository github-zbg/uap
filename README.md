# uap
Angular + NodeJs + Mysql

# Dependencies
Angular 1.4.3
NodeJs (with npm)
hapi.js
Mysql 5.5

# Setup
After cloning, run:
- npm install mysql
- npm install hapi
- npm install inert (serve static files in hapi)
- npm install hapi-auth-cookie

Download js from angularjs.org
- angular-1.4.3.js
- angular-cookies.1.4.3.js
